Selected Data Science & ML Projects

  1. Bayesian Inference of 3D Line Parameters (Metropolis–Hastings MCMC)
    Implemented custom MCMC sampler to infer 3D line endpoints from noisy 2D camera images.
    Used Gaussian priors and multi-view likelihood to achieve accurate MAP reconstruction.
  2. Fairness in Insurance Pricing & Vaccine Misinformation Ethics (Data Ethics)
    Analyzed independence, separation, and sufficiency criteria in ML fairness.
    Conducted ethical analysis on criminalization of vaccine misinformation using real-world scenarios.
  3. Tucson Car Collision Dashboard (Data Visualization – Quarto, R)
    Built interactive dashboard visualizing traffic accidents (2018–2023).
    Used heatmaps, time series, and violation-based statistics to support road safety planning.
  4. Crash Severity Prediction (Data Mining – ML)
    Trained and evaluated Random Forest, Decision Tree, KNN models on Massachusetts crash data.
    Achieved ~78% accuracy; identified environmental and demographic risk factors.
  5. Global Temperature Trends (Time Series + ML)
    Analyzed NASA GISS data (1880–2023) using Mann–Kendall, regression, STL decomposition.
    Applied ML classifiers (XGBoost, RF) to detect and forecast warming anomalies.
  6. Neural Networks & Codabench NLP Competition (Deep Learning)
    Built feedforward NN from scratch (NumPy); Developed Keras architectures (deep vs wide MLP, ReLU vs tanh).
    Implemented CNN and RNN models for MNIST and YouTube spam datasets; Built multi-label NLP classifier using BERT.
  7. Smart Water Distribution and Optimization System (SWDOS) (SQL/DB)
    Designed conceptual–logical–physical database models integrating IoT sensor data.
    Implemented advanced SQL queries: JOINs, subqueries, CASE, aggregation. Built schema for real-time water monitoring and decision support.
