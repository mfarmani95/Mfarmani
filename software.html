Open Source Software Packages

These tools are developed to support hydrologic and climate research. New packages will be added as they are released.

smm-toolkit

A Python package to compute long-term and short-term Soil Moisture Memory (SMM) for hydrological and climate research.

  • Language: Python
  • Initial Release: 2025
PyPI GitHub

bfi-toolkit

A toolkit for Baseflow Index (BFI) analysis and hydrological decomposition of streamflow time series.

  • Language: Python
  • Initial Release: 2025
PyPI GitHub

Upcoming hydrological modeling tools — focused on snowmelt, infiltration, runoff generation, and more.
