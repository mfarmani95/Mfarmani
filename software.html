Open Source Software Packages

I develop and maintain open-source Python packages to support hydrologic research and modeling. These tools are focused on understanding baseflow generation and soil moisture memory, and will be expanded with new modules in the future.

smm-toolkit

A Python package for quantifying short-term and long-term Soil Moisture Memory (SMM) using observational and model datasets. Designed for climate–hydrology research.

  • Language: Python
  • Version: 0.1.0
  • License: MIT
PyPI GitHub

bfi-toolkit

A Python package for computing Baseflow Index (BFI) from streamflow time series using hydrologic separation methods. Optimized for large-scale basin analysis.

  • Language: Python
  • Version: 0.1.0
  • License: MIT
PyPI GitHub
