Open Source Software Packages

These tools are developed to support hydrologic, climate, and data-intensive research. Together they reflect how I work across environmental science, computational modeling, and scientific machine learning.

smm-toolkit

A Python package to compute long-term and short-term Soil Moisture Memory (SMM) for hydrological and climate research.

  • Language: Python
  • Initial Release: 2025
PyPI GitHub

bfi-toolkit

A toolkit for Baseflow Index (BFI) analysis and hydrological decomposition of streamflow time series.

  • Language: Python
  • Initial Release: 2025
PyPI GitHub

Future Tools

Upcoming tools will extend into routing, rainfall-runoff modeling, geospatial preprocessing, and scientific ML workflows for water systems.

What These Packages Represent

  • Hydrology: physically meaningful metrics and process-oriented analysis
  • Water resources: methods that support interpretation, planning, and decision contexts
  • Data science: reusable Python tools and reproducible workflows
  • ML engineering: foundations for benchmarkable, extensible scientific-model pipelines
