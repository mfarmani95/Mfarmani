Open Source Software Packages

Below are open-source Python packages developed as part of my hydrology research. These tools aim to make advanced hydrologic analysis more accessible to the community and are continuously evolving.

1. smm-toolkit

smm-toolkit provides fast, flexible computation of long-term and short-term Soil Moisture Memory (SMM). It’s designed to support hydrologists and climate scientists in analyzing memory dynamics at scale.

  • 📦 Language: Python
  • 📅 Initial Release: 2025
  • 🌐 Repository: smm-toolkit on PyPI
pip install smm-toolkit

2. bfi-toolkit

bfi-toolkit provides tools for Baseflow Index (BFI) analysis and hydrological decomposition of streamflow time series, supporting large-basin studies.

  • 📦 Language: Python
  • 📅 Initial Release: 2025
  • 🌐 Repository: Coming soon
pip install bfi-toolkit

Future Tools

More hydrological and land surface modeling packages are under development — including advanced process representations for infiltration, snowmelt, and runoff.
