Open Source Software Packages

These tools were developed to support hydrologic research and modeling. They are actively maintained and updated with new features and packages over time.

smm-toolkit

smm-toolkit is designed to compute long-term and short-term Soil Moisture Memory (SMM) efficiently. It supports hydrologists and climate scientists working with large-scale datasets.

  • Language: Python
  • Initial Release: 2025
  • PyPI: smm-toolkit
PyPI GitHub

bfi-toolkit

bfi-toolkit provides tools for Baseflow Index (BFI) analysis and hydrological decomposition of streamflow time series, supporting basin-scale water resource studies.

  • Language: Python
  • Initial Release: 2025
  • PyPI: Coming soon
